# 🍳 Recipe Assistant Chatbot

The **Recipe Assistant Chatbot** is an AI-powered culinary platform that helps users discover, create, and manage recipes based on available ingredients, dietary preferences, and cooking time. It provides personalized recipe recommendations, meal planning, and cooking assistance through an intuitive voice-enabled interface.

---

## 🚀 Features

- ✅ **Voice-Activated Search** – Speak or type ingredients to get instant recipe suggestions  
- ✅ **Smart Ingredient Search** – Find recipes using up to 4 ingredients you already have  
- ✅ **Dietary Preference Filters** – Vegan, gluten-free, keto, low-calorie, and more  
- ✅ **Smart Meal Planner** – Generate personalized weekly meal plans  
- ✅ **Favorites Management** – Save and organize your favorite recipes  
- ✅ **Beautiful Blog Section** – Cooking tips, recipes, and culinary insights  
- ✅ **Responsive Design** – Works perfectly on desktop, tablet, and mobile  

---

## 🎯 How It Works

1. **Search by Ingredients**  
   Enter what's in your kitchen to get recipe ideas  

2. **Select Preferences**  
   Choose dietary restrictions and cooking time  

3. **Get Recommendations**  
   Receive personalized recipe suggestions  

4. **Plan Your Meals**  
   Generate complete daily meal plans  

5. **Cook with AI Guidance**  
   Get step-by-step instructions and tips  

---

## 🛠️ Technologies Used

- **Backend:** Flask (Python)  
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5  
- **AI/ML:** Natural Language Processing for voice recognition  
- **Database:** SQLite / MySQL for user data and recipes  
- **APIs:** Integration for recipe data and nutrition information

## ⚙️ Installation

1. Clone the repository
git clone https://github.com/your-repo/recipe-assistant-chatbot.git

cd recipe-assistant-chatbot

2. Create a virtual environment
python -m venv venv

venv\Scripts\activate

3. Install dependencies
pip install -r requirements.txt

4. Run the Flask app
python app.py

5. Open your browser
http://localhost:5000
